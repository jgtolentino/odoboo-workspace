# odoboo-workspace
Native Odoo + OCA stack with OCR microservice and DO deploy scripts.

## Quick start
```bash
./scripts/db-sync-check.sh --db "$DATABASE_URL"
```

* Deploy: see `infra/do/*.sh` and `infra/do/*.yml`
* OCA minimal core: `scripts/download_oca_minimal.sh`
